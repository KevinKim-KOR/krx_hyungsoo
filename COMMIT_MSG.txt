fix: pyarrow engine 명시 - Parquet 읽기/쓰기 오류 해결

## 문제
- pandas read_parquet/to_parquet에서 engine 미지정
- pyarrow 설치되어 있지만 pandas가 자동 감지 실패
- "Unable to find a usable engine" 오류 발생

## 해결
1. infra/data/loader.py
   - read_parquet(path, engine='pyarrow')
   - to_parquet(path, engine='pyarrow')

2. infra/data/updater.py
   - read_parquet(path, engine='pyarrow')
   - to_parquet(path, engine='pyarrow')

3. test_pyarrow.py 추가
   - pyarrow 설치 확인 스크립트
   - Parquet 읽기/쓰기 테스트

## 검증
```bash
python test_pyarrow.py  # ✅ 통과
python pc/cli.py update --symbols 069500 --date 2024-10-30  # ✅ 정상
```
