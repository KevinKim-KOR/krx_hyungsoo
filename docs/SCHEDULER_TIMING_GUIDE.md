**장 운영 시간**:
- 개장: 09:00

## 📋 스케줄 타이밍 (수정)

### 1. 장 시작 알림 ✅
```
시간: 09:00 (정상)
이유: 장 시작 전 포트폴리오 현황
```

### 2. 장중 급등락 알림 ⚠️
```
현재: 11:00, 14:00
문제: 실시간 데이터 조회 필요
해결: 10:00, 13:00, 15:00 (3회)
```

### 3. EoD 신호 ⚠️
```
현재: 15:40
문제: 데이터 아직 확정 안 됨
해결: 16:00 또는 16:30 (권장)
```

### 4. 레짐 변경 감지 ⚠️
```
현재: 16:00
문제: EoD 데이터 필요
해결: 16:30 또는 17:00 (EoD 이후)
```

## ⚠️ 주의사항

### PyKRX 데이터 지연

PyKRX는 장 마감 후 데이터 확정까지 **10~30분** 소요:
- 15:30 장 마감
- 15:40~16:00 데이터 확정
- **16:00 이후 조회 권장**

### 실시간 데이터

장중 알림은 **네이버 금융 API** 사용 권장:
- PyKRX: 일봉 데이터만 (장 마감 후)
- 네이버: 실시간 시세 (장중)

## 📝 Synology 작업 스케줄러 수정

### EoD 신호 (15:40 → 16:00)

1. 작업 스케줄러 → EoD 신호 선택
2. 편집 → 스케줄 탭
3. 시간: **16:00**으로 변경
4. 저장

### 레짐 변경 (16:00 → 16:30)

1. 작업 스케줄러 → 레짐 변경 선택
2. 편집 → 스케줄 탭
3. 시간: **16:30**으로 변경
4. 저장

### 장중 급등락 (11:00, 14:00 → 10:00, 13:00, 15:00)

1. 기존 작업 삭제 또는 수정
2. 새 작업 3개 생성:
   - 10:00 (오전)
   - 13:00 (오후)
   - 15:00 (장 마감 직전)
