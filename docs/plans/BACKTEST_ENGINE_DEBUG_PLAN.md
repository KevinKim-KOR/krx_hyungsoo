## 2. 디버깅 체크리스트

### 2.1 변동성 계산 (최우선)

- [ ] `core/engine/backtest.py`에서 변동성 계산 로직 확인
- [ ] 일간 수익률 계산 정상 여부
- [ ] 변동성 0일 때 Sharpe 처리 (0 또는 NaN)

```python
# 예상 수정
if volatility == 0 or np.isnan(volatility):
    sharpe_ratio = 0.0
else:
    sharpe_ratio = (mean_return / volatility) * np.sqrt(252)
```

## 3. 디버깅 순서

```
1단계: 변동성 계산 수정
  ↓
2단계: 매도 로직 확인
  ↓
3단계: P/L 계산 복구
  ↓
4단계: 거래비용 적용 확인
  ↓
5단계: 전체 백테스트 재실행 및 검증
```

---

## 4. 검증 기준

정상적인 백테스트 결과 예시:

| 지표 | 정상 범위 |
|------|----------|
| CAGR | 10~30% (1년 이상 기간) |
| Sharpe | 0.5~2.0 |
| MDD | 5~20% |
| 변동성 | 10~25% |
| 손익비 | 1.0~3.0 |
| 거래비용 | 0이 아닌 값 |
