## 📊 **매매 신호 명확화**

### **Q1: 매매 신호는 어떤 종목에 대한 신호인가?**
```
A: 보유 종목 기반 신호입니다.

- 매수 신호: 현재 보유하지 않은 종목 중 MAPS 점수 높은 종목
- 매도 신호: 현재 보유 중인 종목 중 손절/리밸런싱 대상

예시:
🟢 매수 추천: KODEX 200 (MAPS: 85.5)
  → 현재 보유하지 않음, 내일 매수 검토

🔴 매도 추천: SK증권 (손절 기준 도달)
  → 현재 보유 중, 내일 매도 검토
```

### **Q2: 내일 매수/매도를 해야 하는가?**
```
A: 현재 파라미터 기준 추천이며, 최종 판단은 사용자가 합니다.

신호 의미:
- "매수 추천" = 현재 전략 기준 매수 타이밍
- "매도 추천" = 현재 전략 기준 매도 타이밍

액션 가이드:
1. 매수 신호 있음
   → 내일 장 시작 후 매수 검토
   → 현재가, 시장 상황 확인 후 결정

2. 매도 신호 있음
   → 내일 장 시작 후 매도 검토
   → 손절 기준, 수익률 확인 후 결정

3. 신호 없음
   → 현재 보유 종목 유지
   → 관망 또는 추가 분석
```

### **Q3: 파라미터는 무엇인가?**
```
A: 백테스트로 최적화된 전략 설정입니다.

현재 파라미터:
- 손절 기준: -15%
- MA 기간: 50/200일
- 레짐 임계값: ±2%
- 포지션 비율:
  · 상승장: 100~120%
  · 중립장: 80%
  · 하락장: 40~60%

Phase 3에서 최적화 예정:
- 실제 보유 종목 백테스트
- 파라미터 최적화 (Optuna)
- 실전 vs 백테스트 비교
```

