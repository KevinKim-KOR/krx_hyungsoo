## 💡 **주의사항**

### **1. 시장가 주문 사용**
```
이유:
- 빠른 체결 필요
- 손절은 가격보다 타이밍이 중요
- 지정가는 미체결 위험

주의:
- 시초가 급락 시 손실 확대 가능
- 거래량 부족 시 체결 지연 가능
```

### **2. 감정적 판단 배제**
```
❌ "조금만 더 기다리면 회복할 것이다"
❌ "손실 확정하기 싫다"
❌ "이미 많이 떨어졌으니 더 떨어지지 않을 것이다"

✅ 손절 기준 기계적 적용
✅ 백테스트 결과 신뢰
✅ 추가 손실 방지 우선
```

### **3. 재매수 조건**
```
손절 후 재진입 조건:
1. 기술적 반등 확인
   - RSI < 30 → > 50
   - MACD 골든크로스
   - 거래량 증가 (5일 평균 대비 2배)

2. MAPS 점수 양전환
   - 음수 → 양수 전환
   - 5점 이상 (강한 신호)

3. 레짐 변경
   - 하락장 → 중립장/상승장 전환
```
