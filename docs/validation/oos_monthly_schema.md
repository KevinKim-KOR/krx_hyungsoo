# OOS Monthly Validation Report Schema (v1.0)

This document defines the JSON structure for `reports/validation/oos_2024_2025_monthly.json`.
This file is generated by `tools/verify_oos_2024_2025.py` and consumed by the UI Dashboard (Validation Tab).

## Schema Overview

```json
{
  "schema_version": "OOS-MONTHLY-1.0",
  "generated_at": "ISO-8601 Timestamp",
  "years": {
    "2024": {
      "period": {
        "start": "YYYY-MM-DD",
        "end": "YYYY-MM-DD"
      },
      "summary": {
        "total_return_pct": float,  # Percentage (e.g. -10.5)
        "mdd_pct": float,           # Positive Float (e.g. 11.2)
        "trades": int,
        "exposure_ratio": float     # 0.0 ~ 1.0
      },
      "monthly": [
        {
          "month": "YYYY-MM",       # e.g. "2024-01"
          "return_pct": float,
          "mdd_pct": float,
          "trades": int,
          "trade_days": int,        # Total trading days in month
          "exposure_ratio": float
        },
        ...
      ]
    },
    "2025": { ... }
  }
}
```

## Field Definitions

### Root Object
- `schema_version`: Version identifier (fixed: "OOS-MONTHLY-1.0").
- `generated_at`: Time when the report was created.
- `years`: Dictionary mapping year string ("2024", "2025") to `YearObject`.

### YearObject
- `period`: Start and End dates of the simulation for this year.
- `summary`: Aggregate metrics for the entire year.
- `monthly`: List of `MonthObject`, sorted by date.

### MonthObject
- `month`: Month identifier (YYYY-MM).
- `return_pct`: Monthly return in percentage.
- `mdd_pct`: Maximum Drawdown within that month.
- `trades`: Number of trades executed (entry) in that month.
- `trade_days`: Total number of market days in that month.
- `exposure_ratio`: Ratio of days with active position / trade_days.
