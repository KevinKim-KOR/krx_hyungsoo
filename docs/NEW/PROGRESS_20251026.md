# 프로젝트 진행 현황 (2025-10-26)

## 1. 현재까지 진행사항

### 1.1 디렉토리 구조 설계
- Clean Architecture 기반 레이어 분리 완료
  ```
  app/        # 애플리케이션 진입점
  core/       # 도메인 로직
  extensions/ # PC 전용 기능
  infra/     # 외부 의존성
  ```

### 1.2 핵심 모듈 작성 시작
- `core/strategy/rules.py`: 전략 규칙 및 HOLD_CORE 정의
- `core/engine/scanner.py`: 시장 스캐너 엔진
- `app/cli/alerts.py`: 알림 CLI 진입점
